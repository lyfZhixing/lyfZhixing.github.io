可能会提示一大堆Untracked文件
Git在未进行commit操作之前，存在三种状态：Untracked files，Changes not staged for commit及Changes to be committed。
Untracked files就是你必须把某些文件放到Git工作目录中，但又不能提交它们，比如保存了数据库密码的配置文件。
可以在.gitignore文件中设置忽略。
